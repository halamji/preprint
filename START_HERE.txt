⚡ 最快 5 分钟上线：

1. 在 GitHub 上创建新 public 仓库（名称：preprint）

2. 本地初始化 Git：
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/preprint.git
   git push -u origin main

3. 启用 GitHub Pages：
   Settings → Pages → Branch: main, Folder: / (root) → Save

4. 修改占位符（在 index.html 和 sitemap.xml）：
   - 将 "yourusername" 改成你的 GitHub 用户名

5. 添加论文：
   python3 add_paper.py --title "My Paper" --authors "Me" --pdf paper.pdf

6. 提交：
   git add papers/ sitemap.xml
   git commit -m "Add first paper"
   git push origin main

✅ 完成！你的论文库现在在线了！

=====================================================================

📊 使用 Python 脚本快速添加论文（推荐）：

python3 add_paper.py \
  --title "深度学习在计算机视觉中的应用" \
  --authors "张三,李四" \
  --abstract "本论文研究了..." \
  --pdf ~/my_paper.pdf \
  --keywords "深度学习,计算机视觉"

脚本会自动：
✅ 创建论文目录
✅ 复制 PDF 文件
✅ 生成 HTML 页面
✅ 更新 sitemap.xml
✅ 更新 papers-manifest.json

🛠️ 管理脚本命令：

./manage.sh clean             清理临时文件
